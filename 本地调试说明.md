# 本地调试说明

## 🚀 一键启动

### macOS/Linux 用户：
```bash
./start.sh
```

### Windows 用户：
双击运行 `start.bat` 或在命令行运行：
```cmd
start.bat
```

### 手动启动（所有系统）：
```bash
# 1. 首次运行需要安装依赖
npm install

# 2. 启动开发服务器
npm run dev
```

---

## 📖 访问网站

启动后，开发服务器通常会在以下地址运行：
- **本地地址**：http://localhost:5173
- **网络地址**：http://[你的IP]:5173

浏览器会自动打开，或者手动在浏览器中访问上述地址。

---

## 🔍 查看团队成员页面

启动后访问以下路径查看团队成员信息：
- 主页：http://localhost:5173
- 团队页面：http://localhost:5173/#/people （具体路径取决于路由配置）

---

## 🛑 停止服务器

在终端中按 `Ctrl + C` 即可停止开发服务器。

## ⚠️ 常见问题

### 1. 端口被占用
如果 5173 端口被占用，Vite 会自动使用下一个可用端口（如 5174）。

### 2. 依赖安装失败
尝试清除缓存后重新安装：
```bash
rm -rf node_modules package-lock.json
npm install
```

### 3. 照片不显示
确保所有照片文件都在：
```
public/img/people/
```
